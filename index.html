Aula de Github

Git init

inicializa o git no projeto

Git add .

lista e adiciona todas as alterações feitas nos arquivos e pastas do projeto ao git e aguara até o envio

git commit -m "message"

adiciona uma mensagem ao histórico de commites no git

Git push

envia os arquivos para o repositório remoto do github

Git status

mostar qual a fase que está sendo executada no git

README

documento importante que descreve o projeto detalhadamente

http://www.readme.so

Visibilidade

para mudar a visiblidade do repositório ->setings (na parte de baixo existe uma area chamada "Danger Zone") ->Change repository visibility

gitkeep

mantem as pastas no status do git mesmo estando vazias

git historico

é onde ficam gravados as alterações feitas no código

Git clone
